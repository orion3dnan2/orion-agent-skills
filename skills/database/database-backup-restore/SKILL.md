# database-backup-restore

## Requirements
- Define logical and physical backup strategy.
- Encrypt backups and restrict access.
- Set retention and off-site copy policy.
- Automate integrity checks.
- Perform scheduled restore drills to an isolated environment.
- Record RPO, RTO, restore commands, ownership fixes, and application verification steps.
