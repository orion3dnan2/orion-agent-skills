# n8n Skills

The reviewed upstream n8n skill pack is maintained at `czlonkowski/n8n-skills`.

Before importing it:

1. Review the current license, scripts, hooks, dependencies, and requested permissions.
2. Pin a reviewed commit or release.
3. Keep credentials out of workflow JSON and repository files.
4. Validate workflows and test error, retry, idempotency, binary-data, AI-agent, and self-hosting behavior.
5. Record the pinned revision in `manifests/upstream-skills.yaml` before production use.
